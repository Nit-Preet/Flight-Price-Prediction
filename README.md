# Flight Price Prediction 

##  Overview
End-to-end ML regression project predicting Indian domestic airline ticket prices (INR).

**Key Highlights:**
- **Best Model**: Random Forest (R² = 0.978, MAE = ₹1,607)
- **Dataset**: ~300k real ticket listings, subsampled to 50k for training
- **Features**: Airline, route, stops, travel class, duration, days until departure
- **Insight**: Business class costs 5–7× Economy; prices spike closer to departure
- **Models Tried**: Linear Regression, Ridge, Lasso, KNN, Decision Tree, Gradient Boosting, Random Forest..

Built with scikit-learn, pandas, Matplotlib, Seaborn. Ready to run!

## ⚡ Quick Start
1. Install dependencies:
   ```

2. Launch Jupyter:
   ```

3. Run notebooks in order: `01` → `02` → `03`

## 🔬 Features

| Feature            | Description                                      |
|:-------------------|:-------------------------------------------------|
| `airline`          | Carrier name                                     |
| `source_city`      | Departure city                                   |
| `destination_city` | Arrival city                                     |
| `departure_time`   | Time bucket (Morning, Evening, etc.)             |
| `stops`            | zero / one / two_or_more → encoded 0 / 1 / 2     |
| `class`            | Economy / Business                               |
| `duration`         | Flight duration in hours                         |
| `days_left`        | Days from booking until departure                |
| `price`            | **Target** — ticket price in INR                 |

## 🔄 Reproducing Results
1. Run `01_eda.ipynb` — explore distributions, correlations, price patterns
2. Run `02_data_cleaning.ipynb` — encode features, export `flights_cleaned.csv`
3. Run `03_model_building.ipynb` — train all models, tune Random Forest, view results
